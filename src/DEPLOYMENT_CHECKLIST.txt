  └─ Go to https://vercel.com

  └─ Configure:

      • Build Command: npm run build
      • Output Directory: dist
      
□ STEP 4: Add Environment Variables
  Copy-paste these in Vercel dashboard → Settings → Environment Variables:

  SUPABASE_SERVICE_ROLE_KEY=eyJ[...your key...]

  OPENAI_API_KEY=sk-[...your key...]
  GROQ_API_KEY=gsk_[...your key...]

  TWILIO_ACCOUNT_SID=AC[...your sid...]
  TWILIO_AUTH_TOKEN=[...your token...]
  TWILIO_PHONE_NUMBER=+1[...your number...]
  
  VAPID_PUBLIC_KEY=BN[...your key...]
  VAPID_PRIVATE_KEY=[...your key...]
  VAPID_SUBJECT=mailto:support@adoras.ai

□ STEP 5: Deploy!
  └─ Click "Deploy" button
  └─ Wait 2-3 minutes for build
  └─ Your app will be live at: https://your-app.vercel.app

□ STEP 6: Test Push Notifications
  └─ Visit your Vercel URL

  └─ Should see: "✅ Service worker registered"
  └─ Go to Settings → Notifications
  └─ Click "Enable Notifications"
  └─ Grant permission when browser asks
  └─ Click "Send Test Notification"
  └─ You should receive a push notification! 🎉
