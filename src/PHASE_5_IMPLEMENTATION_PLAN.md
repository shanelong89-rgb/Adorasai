**Goal:** Detect platform and adapt UI/UX accordingly

### iOS Safari:
- ‚ùå No 100vh support (use -webkit-fill-available)
- ‚ùå No persistent storage without user action
- ‚ùå Input zoom on focus (prevent with font-size: 16px)
- ‚ùå Bounce scrolling can interfere with gestures
- ‚ùå Audio requires user interaction to play
- ‚úÖ Excellent PWA support
- ‚úÖ Add to Home Screen integration

### Android Chrome:
- ‚úÖ Full PWA support
- ‚úÖ Background sync
- ‚úÖ Web Share API
- ‚ùå Various device manufacturers (Samsung, Pixel, etc.)
- ‚ùå Different notch/cutout implementations

- ‚úÖ Keyboard shortcuts
